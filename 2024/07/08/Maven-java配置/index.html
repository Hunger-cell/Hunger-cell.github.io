    Maven_java配置

    发布时间 : 2024-07-08 10:39
    阅读 :

    1、 安装前环境准备
    Idea 2023.3.2
    Jdk 8
    2、 Maven的下载https://maven.apache.org/index.html

    3、 下载后放到d盘的一个全英文的空文件夹
    4、 验证一下maven是否可以使用 mav -v
    5、 配置本地仓库的位置到D盘
    6、 配置阿里云镜像加速
    网址：https://maven.aliyun.com/mvn/guide

    <mirror>
<id>aliyunmaven</id>
<mirrorOf>*</mirrorOf>
<name>阿里云公共仓库</name>
<url>https://maven.aliyun.com/repository/public</url>
</mirror>

    7、配置jdk版本 可以配置多个

    <profile>
    <id>1.8</id>
    <activation>
        <jdk>1.8</jdk>
        <activeByDefault>true</activeByDefault>
    </activation>
    <properties>
        <maven.compiler.source>1.8</maven.compiler.source>
        <maven.compiler.target>1.8</maven.compiler.target>
     <maven.compiler.compilerVersion>1.8</maven.compiler.compilerVersion>
    </properties>
</profile>

    在创建Maven项目之后要设置一下 去设置搜索maven ，设置用户文件 和 仓库目录 就OK了
